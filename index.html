1.Let's set up

  1. Download our boilerplate and slides dezip it on your desktop
  2. Start Sublime Text. It will open a new black window.
  3. Drag & drop your unzipped folder in the Sublime Text window.
  4. You are good to go. You have an index.html, a style.cssfile and an images folder

3. CSS for fonts & colors

Link your stylesheet style.css in the head section of your HTML

<head>
  <link href='style.css' rel='stylesheet'>
</head>

Then copy/paste the CSS code below in style.css and fix it cause it's ugly

  • For fonts use Google Fonts
  • For colors use Colorzilla Chrome plugin to pick colors from other websites (Google Chrome extension)
  • You can also find nice colors on Coolors or Color Hunt
/* style.css */
body {
  background: rgb(248,248,248);
  color: green;
  margin: 0px;
  font-size: 25px;
  font-family: Helvetica;
}

  font-family: Courier;

  font-weight: lighter;

}

}

}

  text-decoration: none;
  color: purple;
}

5. Name your tags

  • Name your banner and footer with id like <div id="banner"> and <div id="footer">
  • Name your features with class like <div class="card">and <a class="btn">
  • Then add this nice CSS code
